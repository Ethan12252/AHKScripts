# AHKScripts

## Getting Started

1. **Install [AutoHotkey v2](https://www.autohotkey.com/) or Use winget `winget install -e AutoHotkey.AutoHotkey`**.
2. **Clone or copy** this folder to your PC.
3. **Edit `config.ini`** For WSL profile..., Can use the template generated by `tool.sh genconfig `.
4. **Run `main.ahk`** to start all features.

---

## Features

- **Navigation Mode**  
  - Press `CapsLock` to enter/exit Vim-like navigation and editing mode.
- **Hotstrings**  
  - Type shortcuts like `\maile` to expand to your email address.
- **Mouse Gestures**  
  - Hold and drag the right mouse button left/right to go forward/back (It sends Alt+Arror right or left).
- **Terminal Shortcuts**  
  - `Ctrl+Alt+T`: Open Windows Terminal  
  - `Ctrl+Alt+P`: Open PowerShell  
  - `Ctrl+Alt+U`: Open WSL (profile name from `config.ini`)  
  - `Ctrl+Alt+Y`: Open VS Code  
  - If File Explorer is focused, these open at the current folder.
- **Config Menu**  
  - Right-click the tray icon and select `~Configs~` to adjust fast-forward speeds.
- **Video Fast Forward**  
  - Long-press right arrow for fast-forwarding in PotPlayer/MPC-BE
- **Laptop specific functions**
  = remap copliot to Ctrl+Shift+o in edge (Open bookMark)

---

## Scripts Overview

- `main.ahk` — Main entry, includes all modules.
- `navigation_mode.ahk` — Modal navigation and editing.
- `hotstrings.ahk` — Email/name expansions.
- `mouse_gesture.ahk` — Mouse gesture navigation.
- `launch_terminal.ahk` — Terminal/VS Code launchers.
- `video_fastfoward.ahk` — PotPlayer/MPC-BE fast-forward.
- `ConfigMenu.ahk` — Tray config GUI.

---

## Tips

- Add `main.ahk` to Windows startup for automatic launch.
- Customize hotstrings and navigation keys in their `.ahk` files.
